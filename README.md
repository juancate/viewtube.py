viewtube.py
====================

A python script to playback videos from youtube and other services.

Requirements:
  - Python 2 or 3
  - [youtube-dl](http://rg3.github.io/youtube-dl/)
  - A video player

Using it
--------------------

First, rename `config.sample.json` to `config.json`

Then, to playback a video write:

    $ ./viewer.py [video url]
